# Healthcare Revenue Cycle Analytics Dashboard

## Overview
An end-to-end revenue cycle management (RCM) analytics project analyzing 
120,000 medical insurance claims using Excel and Power BI.

## Dataset
- **Source:** Kaggle — DenialIQ: 120K Medical Claims
- **Tables:** claims_main, denial_labels, payer_rules
- **Period:** 2021–2024

## Tools Used
- Microsoft Excel (SUMIF, SUMIFS, AVERAGEIF, VLOOKUP, Pivot Tables, Slicers)
- Power BI (DAX measures, relationships, interactive dashboard)

## Key Metrics
- Total Billed Revenue: $266.83M
- Recovery Potential from Appeals: $32.51M
- Collection Rate: 52.11%
- Denial Rate: 28.05%
- Appeal Success Rate: 43.09%

## Key Insights
- Commercial PPO has the highest billed revenue at $78.8M
- Coding errors account for 37.57% of all denials
- CO-11 and CO-16 are the top denial reason codes
- Orthopedic Surgery has the highest denied claim volume
- All payers maintain consistent collection rates of 50–53%
